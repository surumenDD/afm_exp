# afm_exp
このリポジトリはボクセル環境craftiumにおいて、様々な深層強化学習アルゴリズムを用いてagentの性能を比較検証するものになっている。
また、下記Referenceの論文を参考に、LLMのような強力なFoundation modelを用いて報酬関数を設計することで性能がどのように向上するかも比較検証する。

## Reference
This repository is inspired by:
- ARM-FM: Automated Reward Machines via Foundation Models for Compositional Reinforcement Learning,
  Roger Creus Castanyer, Faisal Mohamed, Pablo Samuel Castro, Cyrus Neary, Glen Berseth, arXiv (2025)
  Source: https://arxiv.org/abs/2510.14176
